## **Aster Browser** 

A lightweight, fast, and privacy-focused web browser designed for the modern internet. Aster strips away the bloat to give you a clean, distraction-free browsing experience. 

## **Core Features** 

- **Lightning Fast:** Optimized rendering engine built from the ground up for quick page loads. 

- **Privacy First:** Built-in tracker blocking, ad mitigation, and strict data privacy controls out of the box. 

- **Minimalist UI:** Maximum screen real estate dedicated exclusively to the content you care about. 

- **Customizable:** Full support for bespoke custom themes, user scripts, and workspace configurations.
